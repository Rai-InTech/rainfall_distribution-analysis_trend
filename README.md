# rainfall_distribution-analysis_trend
A complete data science project that performs rainfall distribution analysis for India using EDA, heatmaps ,KDE plots, and ML prediction models. It includes visualizations , trend, analysis and a clean notebook for reproductibility.

This project uses Python, Pandas, NumPy, Matplotlib, Seaborn, and ML algorithms to provide clear insights into rainfall behavior across months and years.
